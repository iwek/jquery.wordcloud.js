jquery.wordcloud.js
===================

A Simple Word Cloud Plugin for JQuery

Usage
-----

      <span>boston</span>
      <span>chicago</span>
      <span>new york</span>
      <span>boston</span>
      <span>boston</span>
      <span>chicago</span>
      <span>boston</span>

and then:

    $(function () {
      $('span').tagcloud();
    });
